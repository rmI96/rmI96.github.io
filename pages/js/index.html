Функції

v N1

v N2

var nameFunction = function (e){ /* code */ console.log(e) } nameFunction('Text')

Параметри за замовчуванням

function nameFunction (e = 'text'){ /* code */ console.log(e) } nameFunction('Text')

Повернення значення

function nameFunction (e){ /* code */ return e + 'string' } console.log(nameFunction('empty'))

Багато аргументів

const arr = ['b', 'c'] function nameFunction(...e) { /* code */ console.log(e); } nameFunction('a', ...arr, 'd') /* ['a', 'b', 'c', 'd'] */
